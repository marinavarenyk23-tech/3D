🦴 Модуль 5: Риггінг та анімація

Цей модуль складається з 3 уроків і присвячений створенню скелетної структури (арматури) та анімації персонажів в Blender.

🦴 Урок 1. Створення арматури (Rigging)

Що таке риггінг?

Риггінг (Rigging) – це процес створення скелетної структури (арматури) для 3D-моделі, яка дозволяє керувати її рухами під час анімації. Кожна кістка (bone) впливає на певну частину моделі, деформуючи її при русі.

Створення арматури в Blender

💌 Як створити базову арматуру:
  1. Додайте арматуру: Shift + A → Armature → Single Bone
  2. Перейдіть в Edit Mode (Tab)
  3. Екструдуйте кістки (E) для створення скелета
  4. Перейменуйте кістки (таз, хребет, ноги, руки)
  5. Налаштуйте батьківські зв'язки (Ctrl + P → Keep Offset)
  6. Перевірте в Pose Mode (Ctrl + Tab)

Weight Painting – призначення ваг

Weight Painting – це процес призначення ваг (значень від 0 до 1) для вершин моделі. Вага визначає, як сильно кожна кістка впливає на конкретну вершину. Це критично для коректної деформації під час анімації.

┌─────────────────────────────────────────────────────────────────────────┐ │ WEIGHT PAINTING ПРИКЛАД │ │ │ │ Кістка ліктя Ваги (червоний = 1, синій = 0) │ │ ┌─────────────┐ ┌─────────────────────────────────────────┐ │ │ │ ● │ │ ██████████ │ │ │ │ │ │ │ ██████████ ← 100% вплив (червоний) │ │ │ │ │ │ │ ░░░███████ │ │ │ │ ● │ → │ ░░░░░█████ ← 50% вплив (зелений) │ │ │ │ │ │ │ ░░░░░░░███ │ │ │ │ │ │ │ ░░░░░░░░░░ ← 0% вплив (синій) │ │ │ │ ● │ └─────────────────────────────────────────┘ │ │ └─────────────┘ │ │ │ │ Чим ближче до суглоба, тим більший вплив кістки │ └─────────────────────────────────────────────────────────────────────────┘
💡 Порада: Використовуйте Auto Weights (Ctrl + P → With Automatic Weights) як стартову точку. Потім вручну виправляйте ваги в режимі Weight Paint (Ctrl + Tab в режимі об'єкта).

🔄 Урок 2. Пряма та зворотна кінематика (FK/IK)

📐 Пряма кінематика (FK)

Forward Kinematics – метод, де кожен суглоб обертається незалежно. Позиція дочірніх кісток залежить від батьківських.

  • ✓ Простий для розуміння
  • ✓ Підходить для рук, хвостів
  • ✗ Складний для позиціювання стоп
  • ✗ Кінцева точка не фіксована

🎯 Зворотна кінематика (IK)

Inverse Kinematics – метод, де позиція кінцевої точки (наприклад, стопи) визначає положення всіх вищестоящих суглобів.

  • ✓ Зручний для ніг (стопа на місці)
  • ✓ Кінцева точка фіксована
  • ✗ Складніший в налаштуванні
  • ✗ Потрібен Pole Target

Налаштування IK для ноги

💌 Як додати IK:
  1. Створіть кістки: thigh → leg → foot
  2. Додайте IK Constraint до кістки foot
  3. Target → вкажіть арматуру, Bone → thigh
  4. Chain Length = 2 (від foot до thigh)
  5. Додайте Pole Target (об'єкт для контролю коліна)
  6. Налаштуйте позицію Pole Target

🎬 Урок 3. Ключова анімація (Keyframe Animation)

Що таке ключові кадри?

Keyframes (Ключові кадри) – це кадри в анімації, на яких задані конкретні значення параметрів об'єкта (положення, обертання, масштаб). Blender автоматично створює проміжні кадри між ключовими (інтерполяція).

Graph Editor – тонке налаштування анімації

Graph Editor – це редактор графіків, який дозволяє редагувати інтерполяцію між ключовими кадрами. Тут можна змінювати швидкість руху, додавати ease in/out, циклічність.

💡 Поради для анімації:
  • Використовуйте Cycles Modifier в Graph Editor для повторюваних анімацій (ходьба, біг)
  • Додавайте Ease In/Out (S-curve) для більш природних рухів
  • Використовуйте Auto Keyframe (кнопка з червоним колом) для автоматичного додавання ключових кадрів
  • Для плавних рухів камери використовуйте Interpolation Mode → Bezier

✏️ Практичні завдання до модуля

📝 Завдання 1: Створення арматури

Створіть арматуру для простого персонажа (або використайте готову модель).

  • Створіть скелет з 10+ кісток (таз, хребет, ноги, руки, голова)
  • Встановіть батьківські зв'язки
  • Виконайте Auto Weights для прив'язки моделі
  • Перевірте деформацію в Pose Mode

Детальна лабораторна робота →

📝 Завдання 2: Налаштування IK

Додайте IK констрейнти для ніг персонажа.

  • Створіть IK для кожної ноги (Chain Length = 2)
  • Додайте Pole Targets для контролю колін
  • Переконайтесь, що стопа залишається на місці при русі тазу

📝 Завдання 3: Анімація ходьби

Створіть циклічну анімацію ходьби для персонажа.

  • Створіть 4 основні пози (Contact, Down, Pass, Up)
  • Додайте ключові кадри для кожної пози
  • Налаштуйте криві в Graph Editor
  • Додайте Cycles Modifier для повторення
  • Експортуйте анімацію у відео (MP4)

⚠️ Часті помилки в риггінгу та анімації

❌ Помилка 1: Неправильна орієнтація кісток (кістки дивляться в інший бік)
✅ Рішення: В Edit Mode виділіть кістку → Ctrl + R (повернути) або Ctrl + N (перерахувати напрямок).
❌ Помилка 2: Артефакти деформації через погані ваги
✅ Рішення: Використовуйте Weight Paint режим для ручного виправлення. Червоний = 100% вплив, синій = 0%.
❌ Помилка 3: "Розтягнуті" суглоби при IK
✅ Рішення: Додайте обмеження на кути обертання (Rotation Limits) в Bone Properties → Inverse Kinematics → Limit Rotation.
❌ Помилка 4: Різка, нереалістична анімація
✅ Рішення: В Graph Editor змініть Interpolation з Linear на Bezier. Додайте Ease In/Out для плавних початку та кінця руху.

🔑 Ключові поняття модуля

  • Риггінг (Rigging) – створення скелетної структури для анімації
  • Арматура (Armature) – набір кісток (Bones)
  • Weight Painting – призначення ваг впливу кісток на вершини
  • FK (Forward Kinematics) – пряма кінематика
  • IK (Inverse Kinematics) – зворотна кінематика
  • Pole Target – контролер напрямку коліна/ліктя в IK
  • Keyframe (Ключовий кадр) – кадр з заданими параметрами
  • Walk Cycle – циклічна анімація ходьби
  • Graph Editor – редактор графіків інтерполяції
