✨ Модуль 3: Матеріали та ноди

Цей модуль складається з 3 уроків і присвячений створенню фотореалістичних та стилізованих матеріалів за допомогою нодової системи Blender.

🎨 Урок 1. Вступ до нодів: Principled BSDF

Що таке нодова система?

Ноди (Nodes) – це візуальне програмування матеріалів в Blender. Кожен нод виконує певну функцію (колір, текстура, математика), а з'єднання між ними створюють складні матеріали.

Principled BSDF – універсальний шейдер

Principled BSDF – це головний шейдер Blender, який об'єднує більшість параметрів матеріалів в одному ноді. Він заснований на Disney Principled BRDF і підходить для створення будь-яких матеріалів – від металу до шкіри, від пластику до скла.

Base ColorОсновний колір матеріалу. Для металу – колір відбиття, для діелектрика – дифузний колір.
Metallic0 = діелектрик (пластик, дерево, камінь), 1 = метал. Не використовуйте проміжні значення для реалістичних матеріалів!
RoughnessШорсткість поверхні. 0 = дзеркальний блиск, 1 = матова поверхня.
IOR (Index of Refraction)Показник заломлення (1.45 для скла, 1.33 для води, 2.42 для алмазу).
AlphaПрозорість (0 = повністю прозорий, 1 = непрозорий).
Subsurface ScatteringПідповерхневе розсіювання – ефект для шкіри, воску, молока.
Emission StrengthСила світіння матеріалу (для ламп, екранів, магічних ефектів).
💡 Порада: Для більшості реалістичних матеріалів використовуйте значення Metallic тільки 0 або 1. Проміжні значення мають сенс лише для спеціальних ефектів (наприклад, брудний метал).

🌀 Урок 2. Процедурні текстури

Що таке процедурні текстури?

Процедурні текстури – це текстури, згенеровані математичними алгоритмами, а не растровими зображеннями. Їх переваги: не мають роздільної здатності, легко масштабуються, не займають місця на диску, легко анімуються.

🌊 Noise Texture

Генерує шум Перліна – найуніверсальніша процедурна текстура. Основа для створення хмар, каменю, піску, металу.

  • Scale – масштаб шуму
  • Detail – рівень деталізації
  • Roughness – шорсткість шуму

🧩 Voronoi Texture

Створює візерунок із комірок на основі діаграми Вороного. Ідеально для панцирів, шкіри рептилій, губок, кристалів.

  • Scale – розмір комірок
  • Feature – Distance to Edge (для тріщин)
  • Randomness – випадковість форм

⛰️ Musgrave Texture

Фрактальна текстура для створення мармуру, тріщин, жил, гірських порід, деревних кілець.

  • Scale – масштаб
  • Detail – деталізація
  • Dimension – фрактальна розмірність

ColorRamp – створення масок та градієнтів

ColorRamp – один з найважливіших нодів. Він відображає вхідні значення (від 0 до 1) на задану колірну шкалу. Використовується для:

  • Створення масок (чорно-білі переходи)
  • Градієнтних переходів кольорів
  • Контролю розподілу матеріалів (де більше іржі, де менше)
  • Стилізації (cell shading, cartoon)
💡 Порада: Використовуйте Math nodes (Multiply, Add, Power) для комбінування текстур та створення складних масок. Наприклад, Multiply двох текстур дасть область перетину.

🔧 Урок 3. Розширені ноди: Masking, Fresnel та Node Groups

Fresnel – ефект кута огляду

Fresnel – оптичний ефект, при якому відбиття стає сильнішим під гострими кутами огляду. Використовується для створення реалістичних матеріалів: скло, вода, лаковані поверхні, тканина.

Node Groups – створення власних блоків

Node Groups – це спосіб згрупувати набір нодів в один блок для повторного використання. Це економить час і робить нодове дерево більш організованим.

💌 Як створити Node Group:
  1. Виділіть ноди, які хочете згрупувати
  2. Натисніть Ctrl + G
  3. У Group Input додайте входи (піни) для параметрів
  4. У Group Output виведіть результат
  5. Збережіть групу як asset (Shift + Ctrl + G)

✏️ Практичні завдання до модуля

📝 Завдання 1: Базові матеріали

Створіть три базові матеріали з використанням тільки Principled BSDF:

  • Метал (золото або мідь) – Metallic=1, налаштуйте Roughness
  • Дерево (лаковане) – використовуйте текстуру дерева або процедурну
  • Скло – IOR=1.45, Transmission=1, Roughness=0.1

Детальна лабораторна робота →

📝 Завдання 2: Процедурний матеріал

Створіть складний процедурний матеріал без використання зовнішніх текстур:

  • Іржавий метал (метал + іржа з маскою)
  • Або дерево з річними кільцями (Voronoi + ColorRamp)
  • Або мармур (Musgrave + ColorRamp)
  • Мінімум 10 нодів у дереві

📝 Завдання 3: Node Group

Створіть Node Group для часто використовуваного ефекту:

  • Наприклад, група "Процедурний метал" з параметрами
  • Додайте входи: Scale, Roughness, Metallic Color
  • Використайте групу в декількох матеріалах
  • Збережіть групу як asset для майбутнього використання

📖 Рецепти матеріалів

🥩 Шкіра (Subsurface Scattering)

  • Base Color: рожево-коричневий
  • Subsurface: 0.2-0.5
  • Subsurface Radius: 1.0, 0.5, 0.2 (RGB)
  • Subsurface Color: яскравіший за Base Color
  • Roughness: 0.4-0.6

💎 Дорогоцінне каміння

  • Principled BSDF → Glass BSDF
  • IOR: 2.42 (алмаз), 1.76 (сапфір)
  • Transmission: 1
  • Roughness: 0.05-0.1
  • Додайте дисперсію (Dispersion=0.02)

🌊 Вода

  • Principled BSDF → Glass BSDF
  • IOR: 1.333
  • Transmission: 1
  • Roughness: 0.1-0.3
  • Додайте хвилі через Noise Texture

🔑 Ключові поняття модуля

  • Principled BSDF – універсальний шейдер Blender
  • Node (нод) – візуальний блок програмування матеріалів
  • Процедурна текстура – текстура, згенерована математично
  • ColorRamp – створення градієнтів та масок
  • Fresnel – ефект посилення відбиття під кутом
  • Node Group – група нодів для повторного використання
  • SSS (Subsurface Scattering) – підповерхневе розсіювання світла

📋 Тести для самоперевірки

Питання 1: Чому для реалістичних металів Metallic має дорівнювати 1, а не проміжним значенням?

Показати відповідь

У фізиці матеріал або є металом (провідником), або ні (діелектриком). Проміжні значення Metallic не відповідають реальним матеріалам і створюють нереалістичний вигляд. Виняток – спецефекти (наприклад, космічні кораблі у фентезі).

Питання 2: Яка різниця між Roughness і Specular?

Показати відповідь

Roughness контролює, наскільки розмитими є відблиски (0 = дзеркало, 1 = матовий). Specular контролює силу дзеркальних відблисків для діелектриків (зазвичай залишається 0.5). Для металів Specular не використовується – відбиття контролюється через Base Color.

Питання 3: Навіщо використовувати процедурні текстури замість зображень?

Показати відповідь

Процедурні текстури не мають роздільної здатності (не пікселізуються при наближенні), легко масштабуються, не займають місця на диску, можуть бути анімовані, ідеально тайляться без швів. Основний недолік – складність створення органічних текстур (фотографічна шкіра, дерево).
