🗺️ Модуль 2: UV-мапінг та текстурування

Цей модуль складається з 4 уроків і присвячений створенню UV-розгорток та накладанню текстур на 3D-моделі.

📐 Урок 1. Основи UV-розгортки

Що таке UV-координати?

UV-мапінг – це процес проєкції 3D-моделі на 2D-площину для накладання текстур. UV-координати визначають, якій частині 2D-текстури відповідає кожна грань моделі.

Аналогія: UV-розгортка – це як розрізати паперову модель і розкласти її на площині. Кожен шматок паперу (UV island) потім можна розфарбувати (накласти текстуру).

💡 Порада: UV-координати називаються "U" та "V" замість "X" та "Y", щоб відрізняти їх від 3D-координат простору. Діапазон UV – від 0 до 1.

Основні терміни UV-мапінгу

  • UV Island (острів) – група з'єднаних граней, які разом розгортаються на площину
  • Seam (шов) – позначене ребро, де "розрізається" модель для розгортки
  • Texel Density (щільність текселів) – кількість пікселів текстури на одиницю площі моделі
  • Stretch (розтягнення) – спотворення UV-розгортки відносно оригінальної геометрії

✂️ Урок 2. Ручне розміщення швів (Seams)

Де розміщувати шви?

Правильне розміщення швів – ключ до якісної UV-розгортки. Шви слід розміщувати в непомітних місцях:

👤

Для персонажа

• За головою (потилиця)
• Під пахвами
• Внутрішня сторона ніг
• Підошви стоп

🚗

Для транспорту

• По нижній частині
• В місцях з'єднань
• Всередині колісних арок
• По краях деталей

🏠

Для архітектури

• По кутах будівель
• По краях даху
• На стиках стін
• На невидимих сторонах

💡 Порада: Якщо не знаєте, де краще розмістити шов – уявіть, що ви робите модель з тканини. Там, де ви б зробили розріз або шов, там і маркуйте Seam!

🎨 Урок 3. Текстурування: растрові текстури та Texture Paint

Типи текстур

  • Растрові текстури – звичайні зображення (PNG, JPG, TGA) з фіксованою роздільною здатністю
  • PBR текстури – набір текстур (Albedo, Normal, Roughness, Metallic) для фотореалістичних матеріалів
  • Texture Paint – малювання текстури безпосередньо на 3D-моделі в Blender

Основні налаштування Texture Paint

  • Brush Settings – розмір, сила, жорсткість пензля
  • Color – вибір кольору (палітра або eyedropper)
  • Texture Mask – використання текстур як масок для пензля
  • Stencil – режим трафарету для точного малювання
⚠️ Важливо: Перед використанням Texture Paint переконайтеся, що ви створили текстуру для малювання! В UV Editor: Image → New, задайте розмір (наприклад, 2048×2048).

📤 Урок 4. Експорт UV-схеми та робота із зовнішніми редакторами

Як експортувати UV-схему

1
У режимі Edit Mode виділіть всі грані (A)
2
UV Editor → UV → Export UV Layout
3
Виберіть формат (PNG) та розмір
4
Відкрийте схему в GIMP/Krita/Photoshop
5
Намалюйте текстуру, використовуючи UV-лінії як орієнтир
6
Імпортуйте текстуру назад у Blender (Image Texture)

Baking (запікання) текстур

Baking – це процес перенесення інформації з одного об'єкта на текстуру іншого. Найчастіше використовується для:

  • Normal Map – запікання деталей з high-poly на low-poly модель
  • Ambient Occlusion – запікання затінення
  • Diffuse Map – запікання кольору з матеріалів
💡 Порада для Normal Map baking:
  • High-poly модель має перекривати low-poly
  • Увімкніть Cage (Extrusion) для запобігання артефактам
  • Використовуйте Selected to Active (спочатку виберіть high-poly, потім low-poly)
  • Встановіть Margin (відступ) 8-16 пікселів для чистих країв

✏️ Практичні завдання до модуля

📝 Завдання 1: UV-розгортка транспортного засобу

Створіть UV-розгортку для моделі автомобіля або корабля з правильно розміщеними швами.

  • Розмістіть шви в логічних місцях (стики деталей, нижня частина)
  • Виконайте UV Unwrap
  • Розташуйте islands для оптимального використання простору
  • Перевірте stretch (синій колір = добре, червоний = погано)

Детальна лабораторна робота →

📝 Завдання 2: Texture Paint

Намалюйте текстуру безпосередньо на 3D-моделі в режимі Texture Paint.

  • Створіть нову текстуру (2048×2048)
  • Намалюйте базовий колір та деталі
  • Додайте подряпини, бруд або інші ефекти
  • Використайте Stencil для точного нанесення

📝 Завдання 3: Baking Normal Map

Виконайте baking normal map з high-poly скульптингу на low-poly модель.

  • Створіть high-poly модель з деталями (скульптинг)
  • Створіть low-poly модель з чистою топологією
  • Розгорніть UV для low-poly моделі
  • Виконайте baking Normal map
  • Застосуйте normal map до low-poly моделі

🔑 Ключові поняття модуля

  • UV координати – 2D-координати для накладання текстур
  • Seam (шов) – розріз моделі для UV-розгортки
  • UV Island – група граней, розгорнутих разом
  • Texel Density – щільність пікселів текстури
  • Texture Paint – малювання текстури на 3D-моделі
  • Baking – запікання інформації в текстуру
  • Normal Map – текстура, що імітує деталі геометрії

📋 Тести для самоперевірки

Питання 1: Що означають синій та червоний колір у режимі перевірки stretch UV?

Показати відповідь

Синій колір означає відсутність або мінімальне спотворення UV-розгортки (добре). Червоний колір вказує на значне розтягнення або стиснення UV островів (погано). Ідеальна UV-розгортка має бути переважно синьою або зеленою.

Питання 2: Чому важлива однакова texel density для різних об'єктів сцени?

Показати відповідь

Однакова texel density забезпечує візуальну узгодженість – всі об'єкти будуть мати однакову чіткість текстур. Якщо texel density різна, деякі об'єкти виглядатимуть більш розмитими (низька щільність) або надто деталізованими (висока щільність) відносно інших.

Питання 3: Яка різниця між Smart UV Project та ручною розгорткою?

Показати відповідь

Smart UV Project автоматично створює шви та розгортку на основі кута між гранями. Це швидко, але не завжди оптимально для складних моделей. Ручна розгортка дозволяє контролювати розміщення швів та форму islands, що дає кращий результат для органічних моделей та персонажів.

Питання 4: Для чого потрібен Cage при baking normal map?

Показати відповідь

Cage (клітка) – це зовнішня оболонка, яка визначає, як далеко від low-poly моделі будуть «збиратися» деталі з high-poly. Правильний Cage запобігає артефактам baking та забезпечує чисту normal map без спотворень на краях.

⚠️ Часті помилки та їх вирішення

❌ Помилка 1: Перекриття UV island (один острів накладається на інший)
✅ Рішення: В UV Editor перемістіть islands так, щоб вони не перекривалися. Використовуйте Pack Islands (UV → Pack Islands).
❌ Помилка 2: Сильне розтягнення текстури (stretch)
✅ Рішення: Додайте більше швів та зробіть розгортку заново. Використовуйте Minimize Stretch (UV → Minimize Stretch).
❌ Помилка 3: Низька texel density на великих поверхнях
✅ Рішення: Збільште розмір текстури або використовуйте окремі UDIM тайли для різних частин моделі.
