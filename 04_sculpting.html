🖌️ Модуль 4: Скульптинг

Цей модуль складається з 3 уроків і присвячений цифровому скульптингу – створенню органічних форм за допомогою пензлів (брейшів). Скульптинг ідеально підходить для персонажів, тварин, монстрів, рельєфів та будь-яких органічних об'єктів.

🖌️ Урок 1. Інструменти скульптингу: базові брейші

Що таке скульптинг?

Скульптинг – це метод 3D-моделювання, який імітує роботу з глиною. Замість точного маніпулювання вершинами, художник "ліпить" форму, використовуючи різні пензлі (брейші). Це інтуїтивний спосіб створення органічних форм.

Основні брейші (пензлі) для скульптингу

✏️

Draw

Піднімає поверхню. Основний брейш для створення форми.

Клавіша: Shift + S
🧱

Clay Strips

Створює плавні гребені. Ідеальний для м'язів, складок.

Клавіша: Shift + C
🖐️

Grab

Перетягує великі області. Для грубого формування.

Клавіша: Shift + G
🌀

Smooth

Згладжує поверхню. Виправляє артефакти.

Клавіша: Shift + F
📏

Crease

Створює чіткі лінії, морщини, шви.

Клавіша: Shift + E
🎈

Inflate

Роздуває поверхню. Для об'ємних форм.

Клавіша: Shift + I

💡 Порада: Змінюйте розмір та силу пензля клавішами F (розмір) та Shift + F (сила) під час малювання. Це пришвидшує роботу в рази!

🌀 Урок 2. Динамічна топологія (Dyntopo) та Remesh

Що таке Dyntopo?

Dyntopo (Dynamic Topology) – режим скульптингу, який динамічно додає полігони там, де це необхідно (в місцях деталізації). Він дозволяє починати з дуже простої моделі та додавати полігони лише там, де потрібна деталізація.

Налаштування Dyntopo

  • Detail Size (Detect) – розмір деталізації в пікселях. Менше число = більше полігонів.
  • Detail Type – тип деталізації: Constant (фіксована), Brush (залежить від пензля), Relative (відносна до масштабу).
  • Refine Method – Subdivide Edges (поділ ребер) або Collapse Edges (видалення полігонів при зменшенні).

Remesh – перебудова топології

Remesh – інструмент для перебудови всієї топології скульптингу з однаковим розміром полігонів (воксельний ремеш). Використовується перед ретопологією або для вирівнювання щільності сітки.

💌 Як використовувати Dyntopo:
  1. Увімкніть Dyntopo в верхній панелі (іконка каменя)
  2. Встановіть Detail Size (почніть з 10-15 px)
  3. Почніть скульптинг – полігони будуть додаватися автоматично
  4. Для більшої деталізації зменште Detail Size (4-8 px)
  5. Використовуйте Remesh (Ctrl + R) для повної перебудови сітки
⚠️ Важливо: Dyntopo генерує трикутники (tris), а не чотирикутники (quads). Це нормально для скульптингу. Для анімації знадобиться ретопологія (чисті quads).

✨ Урок 3. Альфа-брейшинг та деталізація

Що таке альфа-брейшинг?

Альфа (Alpha) – це текстура у форматі grayscale (чорно-біла), яка визначає форму відбитка пензля. Альфа-брейшинг дозволяє створювати складні деталі одним рухом: луска, шкіра, тріщини, клепки, броня.

Де взяти альфа-текстури?

  • Вбудовані альфи Blender – Textures Properties → Brush → Texture
  • Безкоштовні бібліотеки – BlendSwap, ArtStation (деякі безкоштовні), OpenGameArt
  • Створення власних – малювання в GIMP/Krita/Photoshop (grayscale, PNG)

Stencil – режим трафарету

Stencil – режим, де альфа текстура проєктується на модель через спеціальний трафарет. Це дозволяє точно розміщувати деталі, наприклад, татуювання, шрами, логотипи.

💡 Порада для Stencil:
  • Увімкніть Stencil (Brush Settings → Texture → Mapping → Stencil)
  • Переміщуйте трафарет: R (обертання), S (масштаб), Ctrl + F (скинути позицію)
  • Малюйте, поки трафарет активний

📋 Етапи створення скульптингу

1
Блокування (Blockout) – створення базової форми за допомогою Grab, Clay Strips. Не турбуйтесь про деталі.
2
Основна форма (Primary Form) – додавання основних рис: голова, тулуб, кінцівки. Використовуйте Draw, Inflate.
3
Вторинні форми (Secondary Form) – м'язи, складки, суглоби. Clay Strips, Crease.
4
Деталізація (Tertiary Form) – дрібні деталі: шкіра, пори, тріщини, луска. Альфа-брейші, Stencil.
5
Фінальна обробка – згладжування артефактів (Smooth), перевірка симетрії.

✏️ Практичні завдання до модуля

📝 Завдання 1: Скульптинг голови

Створіть скульптинг голови людини або фантастичної істоти з використанням базових брейшів та симетрії.

  • Почніть зі сфери або куба
  • Увімкніть X-axis mirror (симетрію)
  • Використовуйте Grab для грубої форми
  • Додайте очні западини, ніс, рот, вуха
  • Використовуйте Smooth для згладжування

Детальна лабораторна робота →

📝 Завдання 2: Деталізація з Dyntopo

Додайте деталізацію до вашої моделі за допомогою Dyntopo.

  • Увімкніть Dyntopo з Detail Size 10 px
  • Зменште розмір до 5 px для дрібних деталей
  • Додайте текстуру шкіри (альфа-брейш)
  • Використайте Crease для зморшок

📝 Завдання 3: Альфа-брейшинг

Створіть модель з повторюваними деталями (луска, броня, клепки) за допомогою альфа-брейшів.

  • Знайдіть або створіть альфа-текстуру луски/броні
  • Завантажте альфу в налаштування пензля
  • Нанесіть деталі на поверхню моделі
  • Спробуйте режим Stencil для точного розміщення

⚠️ Часті помилки в скульптингу

❌ Помилка 1: Занадто маленький розмір деталізації Dyntopo з самого початку
✅ Рішення: Починайте з великого розміру (15-20 px) для грубої форми. Зменшуйте розмір поступово, коли потрібна деталізація.
❌ Помилка 2: Ігнорування масок (Mask)
✅ Рішення: Використовуйте маски для захисту вже готових областей. M – малювати маску, Alt + M – очистити.
❌ Помилка 3: Занадто сильне використання Smooth
✅ Рішення: Smooth зменшує деталізацію. Використовуйте його тільки для виправлення артефактів.

🔑 Ключові поняття модуля

  • Скульптинг – метод 3D-моделювання "ліпленням" пензлями
  • Брейш (Brush) – пензель з певною формою та функцією
  • Dyntopo – динамічна топологія, додавання полігонів при малюванні
  • Remesh – повна перебудова топології з однаковими полігонами
  • Альфа (Alpha) – текстура для форми відбитка пензля
  • Stencil – режим трафарету для точного проєктування альфи
  • Mask – захист областей моделі від змін
  • Symmetry (Mirror) – симетричний скульптинг по осі X
