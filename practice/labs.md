# 🧪 Лабораторні роботи з 3D моделювання

Цей розділ містить 6 лабораторних робіт – по одній на кожен модуль курсу. Кожна лабораторна робота розрахована на 2-4 академічні години та передбачає створення завершеного 3D-об'єкта або сцени.

---

## 📋 Загальні вимоги до виконання лабораторних робіт

1. **Створіть окрему папку** для кожної лабораторної роботи у форматі: `lab_XX_nazva/`
2. **Зберігайте файли** у форматі `.blend` з осмисленими назвами
3. **Експортуйте рендер** у форматі PNG (1920×1080 або 4K для фінального проекту)
4. **Додайте скріншоти** процесу роботи (2-3 проміжні етапи)
5. **Напишіть короткий звіт** (text або markdown файл) з описом використаних технік

## Лабораторна робота №1: Полігональне моделювання зброї

### Мета
Застосувати на практиці модифікатори Mirror, Solidify та Bevel для створення симетричної моделі холодної зброї.

### Тривалість
3 академічні години

### Завдання

Створіть модель **середньовічного меча або сокири**, використовуючи наступні техніки:

#### Базові вимоги:
- [ ] Використання модифікатора **Mirror** для симетрії
- [ ] Використання модифікатора **Solidify** для клинка
- [ ] Використання модифікатора **Bevel** на гострих кутах
- [ ] Мінімум **3 різних матеріали** (клинок, руків'я, гарда)
- [ ] **Loop cuts** для додавання деталей руків'я

#### Додаткові вимоги (для вищої оцінки):
- [ ] Складний ефес (гарда) з кількома елементами
- [ ] Деталізація руків'я (оплітка, дорогоцінне каміння)
- [ ] Гравіювання або орнамент на клинку
- [ ] Правильна топологія (тільки quads, без N-gons)

### Покрокова інструкція


### Критерії оцінювання

| Критерій | Бали | Опис |
|----------|------|------|
| Симетрія та пропорції | 10 | Правильне використання Mirror |
| Топологія | 10 | Чисті quads, правильні edge loops |
| Деталізація | 10 | Кількість та якість деталей |
| Матеріали | 10 | Реалістичність, різноманітність |
| Загальне враження | 10 | Естетика, креативність |

**Максимум: 50 балів**

### Результат лабораторної роботи

Надайте:
1. Файл `.blend` з моделлю
2. Рендер у форматі PNG (1920×1080)
3. Скріншоти 3 проміжних етапів
4. Текстовий звіт (використані техніки, складності)

## Лабораторна робота №2: UV-розгортка та текстурування транспортного засобу

### Мета
Навчитися створювати UV-розгортку складного об'єкта та накладати растрові текстури.

### Тривалість
3 академічні години

### Завдання

Створіть UV-розгортку та затекстуруйте **ретро-автомобіль** або **космічний корабель**.

#### Базові вимоги:
- [ ] Модель має мати **10+ об'єктів** (кузов, колеса, вікна, фари)
- [ ] Кожен об'єкт має власну **UV-розгортку**
- [ ] Створення **швів (seams)** у логічних місцях (пахви, кути)
- [ ] Використання **Texture Paint** для малювання текстури прямо на моделі
- [ ] Накладання **Image Texture** через Principled BSDF

#### Додаткові вимоги (для вищої оцінки):
- [ ] Налаштування **texel density** (однакова щільність для всіх об'єктів)
- [ ] **Baking** нормалей з high-poly деталей
- [ ] Створення **маски іржі** або подряпин
- [ ] UV islands без перекриттів та з мінімальним спотворенням

### Покрокова інструкція

### Критерії оцінювання

| Критерій | Бали | Опис |
|----------|------|------|
| Якість UV-розгортки | 15 | Мінімальні спотворення, логічні шви |
| Використання простору | 10 | Оптимальне розміщення islands |
| Якість текстури | 15 | Деталізація, реалістичність |
| Texture Paint | 10 | Вміння малювати на моделі |

**Максимум: 50 балів**

## Лабораторна робота №3: Процедурні матеріали

### Мета
Опанувати створення складних процедурних матеріалів без використання зовнішніх текстур.

### Тривалість
2 академічні години

### Завдання

Створіть **3 різних процедурних матеріали** для демонстраційної сцени:

1. **Іржавий метал** (поєднання металу та іржі з маскою)
2. **Дерево з річними кільцями**
3. **Скло з бульбашками/тріщинами**

#### Базові вимоги:
- [ ] Жодних зовнішніх текстур (тільки процедурні ноди)
- [ ] Мінімум **10 нодів** на матеріал
- [ ] Використання **ColorRamp** для створення масок
- [ ] Використання **Math nodes** для комбінування текстур
- [ ] Налаштування **Roughness**, **Metallic**, **Specular**

#### Додаткові вимоги (для вищої оцінки):
- [ ] Створення **Node Group** для повторного використання
- [ ] Анімація параметрів текстури (рух хвиль, зміна кольору)
- [ ] Матеріал з **субповерхневим розсіюванням (SSS)**
- [ ] Матеріал з **дисплейсментом** (рельєфом)
