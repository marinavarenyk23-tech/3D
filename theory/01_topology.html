📖 Урок 1. Анатомія полігону: вершини, ребра, грані, нормалі

Основні поняття

🔄 Урок 2. Топологія для деформації: Edge Loops та правильна структура

Що таке правильна топологія?

Правильна топологія – це організація полігональної сітки, яка забезпечує коректну деформацію під час анімації та згладжування. Основне правило: використовуйте чотирикутники (quads), уникайте трикутників (tris) та багатокутників (N-gons) там, де модель буде деформуватися.

Edge Loops (Петлі ребер)

Edge Loop – це послідовність ребер, що йдуть вздовж поверхні моделі, де кожне наступне ребро з'єднується з попереднім через вершини. Вони критично важливі для:

  • Деформації обличчя (очиці, рот)
  • Згинання кінцівок (лікті, коліна)
  • Додавання деталей без порушення структури
💡 Порада: Використовуйте Ctrl + R для створення Loop Cuts. Це найшвидший спосіб додати деталізацію, не порушуючи топологію.

⚙️ Урок 3. Модифікатори: Mirror, Subdivision Surface, Solidify

Модифікатори як non-destructive інструменти

Модифікатори – це інструменти, які змінюють об'єкт без редагування його базової геометрії. Їх можна додавати, видаляти, змінювати порядок – це дає гнучкість у роботі.

🔄 Урок 5. Ретопологія вручну

Що таке ретопологія і коли вона потрібна?

Ретопологія – це процес створення чистої, оптимізованої полігональної сітки поверх скульптингу з високою деталізацією. Вона необхідна для:

  • Анімації – чиста топологія забезпечує правильну деформацію
  • Оптимізації – зменшення кількості полігонів для ігор
  • UV-розгортки – на чистій топології легше створювати шви
  • Baking нормалей – перенесення деталей з high-poly на low-poly

Інструменти для ретопології в Blender

  • Shrinkwrap Modifier – прилипання нової сітки до high-poly моделі
  • Poly Build Tool – ручне створення полігонів (Shift + R)
  • BSurfaces Addon – створення сітки по кривих
  • Snapping (прив'язка) – Face Project + Align Rotation to Target
💡 Поради для ретопології:
  • Почніть з найбільш критичних зон (обличчя, суглоби)
  • Дотримуйтесь анатомічних ліній (edge loops навколо очей, рота)
  • Використовуйте симетрію (X-axis mirror), якщо модель симетрична
  • Перевіряйте щільність полігонів – більше деталей потребують більше полігонів

✏️ Практичні завдання до модуля

📝 Завдання 1: Створення простої моделі з Mirror

Створіть модель меча або сокири з використанням модифікатора Mirror.

  • Використовуйте Mirror по осі X
  • Додайте Solidify для клинка
  • Додайте мінімум 3 різних матеріали
  • Зробіть рендер результату

Детальна лабораторна робота →

📝 Завдання 2: Моделювання з Array та Boolean

Створіть колону або паркан з використанням Array та Boolean модифікаторів.

  • Створіть один сегмент
  • Використайте Array для повторення
  • Додайте Boolean для створення отворів або з'єднань
  • Застосуйте Subdivision Surface для згладжування

📝 Завдання 3: Ретопологія простої форми

Виконайте ретопологію простого скульптингу (наприклад, яблука або каменя).

  • Заскільптить просту форму
  • Створіть новий об'єкт для ретопології
  • Додайте Shrinkwrap modifier
  • Створіть чисту сітку вручну (Poly Build)

🔑 Ключові поняття модуля

  • Топологія – структура полігональної сітки
  • Edge Loop – петля ребер для деформації
  • Pole (полюс) – вершина, де сходяться 5+ ребер
  • Non-destructive modeling – використання модифікаторів
  • Ретопологія – створення чистої сітки поверх високодеталізованої
  • Нормаль – вектор, перпендикулярний до поверхні

📋 Тести для самоперевірки

Питання 1: Чому трикутники (tris) та N-gons вважаються поганою топологією для анімованих моделей?

Показати відповідь

Трикутники та N-gons викликають артефакти під час деформації та роботи модифікатора Subdivision Surface. Вони створюють непередбачувані складки та «шпилі» при згинанні моделі. Чотирикутники (quads) забезпечують передбачувану деформацію і краще згладжуються.

Питання 2: Яка різниця між застосуванням (Apply) та залишенням модифікатора в стеку?

Показати відповідь

Залишення модифікатора дозволяє повернутися до редагування параметрів будь-коли (non-destructive). Застосування модифікатора «запікає» зміни в геометрію, після чого їх не можна скасувати. Застосовуйте модифікатори лише коли впевнені, що більше не потрібно буде їх налаштовувати.

Питання 3: Навіщо робити ретопологію, якщо можна використовувати Dyntopo?

Показати відповідь

Dyntopo створює дуже щільну та хаотичну топологію, яка не підходить для анімації, UV-розгортки та використання в іграх. Ретопологія дозволяє отримати чисту, оптимізовану сітку з quads, яка коректно деформується та займає менше пам'яті.
